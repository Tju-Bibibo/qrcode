#qrcode使用指南

###可执行程序使用方法
clone qrcode_tkinter.exe程序到本地，然后直接即可运行，按照指示选择自定义的背景和参数即可
###网页版使用方法
网页版链接————>[qrcode](http://bibibo.net:5000)
网页版可能运行速度比可执行程序慢，上传的背景图片最大不能超过1M
注意网页版的转换的文本和链接是分开的，二者只能二选一
**小站勿压谢谢**
